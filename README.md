<h1 align="center" id="title">Transformer Text Style Prediction</h1>

<h2>üßê Features</h2>

Here're some of the project's best features:

*   Transformer-based text style predictor
*   Simple web interface to test the model (index.html)
*   Custom training on any text you provide
*   Outputs text in the learned style

```
git clone https://github.com/aditya9515/Transformer_Text_Style_Prediction.git     cd Transformer_Text_Style_Prediction
```

<p>2. Start the Flask backend</p>

```
 python main.py
```

```
start intex.html
```

<p>4. Open the web interface</p>

```

```

```

```
